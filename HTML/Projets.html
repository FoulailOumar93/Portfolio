2025 – CinéDélices (Projet O’Clock)

CinéDélices est une application web full-stack créée durant ma formation chez O’Clock. Elle associe recettes de cuisine et univers du cinéma.

Objectifs

  • Créer une vraie application web complète (front + back).
  • Gérer recettes, ingrédients, médias, utilisateurs.
  • Permettre CRUD complet + upload images.
  • Interface moderne et responsive.

Mon rôle

  • Développement complet du back-end Node.js / Express.
  • Création des modèles PostgreSQL + relations Sequelize.
  • Authentification JWT.
  • Développement du front React (pages, composants, routing).
  • Upload d’images avec FormData.
  • Création de la charte graphique (UI/UX).
  • Déploiement sur VM O’Clock.

Stack utilisée

  • Front-end : React, React Router
  • Back-end : Node.js, Express, Sequelize
  • BDD : PostgreSQL
  • Auth : JWT

Difficultés rencontrées

  • Relations Sequelize (1-N, N-N).
  • CORS entre front et back.
  • Gestion d’upload images + preview.
  • Responsive design.

Solutions

  • Middleware CORS personnalisé.
  • Refactor modèles Sequelize.
  • Gestion d’erreurs front + loaders.
  • Refonte de plusieurs pages pour rendre l’UX fluide.
